socket 0456
packet format:
00: IPX Header
 (1E: SPX Header)
2A: component position
2B:  request type
2C:  additional data

components:
00  IPX/SPX
01  bridge driver
02  shell driver
03  shell
04  VAP shell
05  external bridge
06  internal bridge
07  nondedicated NetWare server
08  Star 68000
09  DOS GNMA
0a  OS2 GNMA
0b  NWlite Server
0c  NWlite Cache

fileservers : usually components 0,1,6
workstations: usually components 0,2,3,9
